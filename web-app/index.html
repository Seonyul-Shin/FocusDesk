새 과제 추가

Chrome 확장 프로그램 연동 안내

노트북에서 활성화된 탭 주소 및 지속시간을 실시간으로 추적하려면 Chrome 확장이 필요합니다.

  1. 확장 프로그램 폴더 준비

    프로젝트 폴더 내 /extension 폴더를 로컬 디렉토리에 둡니다.

  2. 크롬 확장 페이지 이동

    크롬 주소창에 chrome://extensions를 입력하고 엔터를 누릅니다.

  3. 개발자 모드 활성화

    우측 상단의 개발자 모드 스위치를 켭니다.

  4. 패키지 되지 않은 확장 로드

    좌측 상단의 [압축해제된 확장 프로그램을 로드합니다] (Load unpacked) 버튼을 클릭합니다.

  5. 폴더 선택

    컴퓨터 내에 준비된 /extension 폴더를 통째로 선택합니다.

  6. 연동 키(Sync Key) 설정

    웹 화면 우측 상단이나 대시보드에 나타나는 고유 User UID를 복사한 후, 크롬 확장 아이콘을 클릭하여 나타나는 팝업에 입력하고 [연동하기] 버튼을 누르면 연동이 끝납니다.
